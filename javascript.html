If you have no idea what I am talking about, then either ask somebody to help you or perform a websearch for "Turning Javascript On/Off".

Don't Worry Though

The random Article feature is not a site-critical feature, so don't worry if you prefer to leave Javascript off. The site will function perfectly well without it. You just lose the mystery of finding something you might never search for.
